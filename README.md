## Run Locally

Clone the project

```bash
  git clone https://github.com/Sumit240803/full-stack-internship-assignment.git
```





Run the server(Change the port according to your local PHP server)

```bash
  php -S localhost:8080
```
Run HTML
```bash
  localhost:5500/index.html
```
